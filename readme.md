# is-up-cli

> Check whether a website is up or down using the [isitup.org](https://isitup.org) API

<img src="screenshot.png" width="833">

## Install

```sh
npm install --global is-up-cli
```

## Usage

```
$ is-up --help

  Example
    $ is-up sindresorhus.com
    âœ” Up

  Exits with code 0 if up and 2 if down
```
